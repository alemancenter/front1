# Frontend AI Model Router UI

تمت إضافة اختيار "استراتيجية الموديلات" داخل مركز تحسين جودة المحتوى في:

```txt
/dashboard/content-audit
```

الخيارات:

- اقتصادي: مناسب للفحص الجماعي الواسع.
- متوازن: مناسب لمعظم المعالجات.
- جودة عالية: مناسب للمحتوى المهم.
- مراجعة نهائية: مناسب لأهم الصفحات قبل الاعتماد أو إعادة تقديم AdSense.

يتم إرسال القيمة إلى الباك اند عبر `model_strategy` ضمن طلب إنشاء مهمة المعالجة الجماعية.
