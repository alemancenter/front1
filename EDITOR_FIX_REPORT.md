# Editor Fix Report

تم إصلاح محرر المقالات والمنشورات بعد ملاحظات الاختبار:

- إصلاح سبب عدم عمل أزرار المحرر عبر حفظ واسترجاع selection ومنع فقدان التركيز عند الضغط على أزرار toolbar.
- توسيع أدوات المحرر: نمط النص، حجم الخط، ألوان النص والخلفية، محاذاة كاملة، إزالة الرابط، خط فاصل، جدول، صورة، ملف، تراجع/إعادة.
- إصلاح مشكلة الصور التي تُدرج برابط داخلي مثل `http://127.0.0.1:8080/storage/...` عبر تحويلها إلى `/storage/...` حتى تعمل من نفس الدومين وتدخل ضمن CSP `self`.
- إضافة استثناءات CSP مؤقتة لعرض الصور القديمة التي قد تكون محفوظة بروابط localhost أثناء التطوير.
- لا توجد اعتماديات npm جديدة.
- لا يوجد تعديل في طريقة حفظ المحتوى: ما يزال الحفظ HTML داخل `content`.
