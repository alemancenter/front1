# Text Editor Migration Report

تم استبدال Summernote/jQuery بمحرر React خفيف يعتمد على `contentEditable` وأوامر المتصفح الأساسية، بدون إضافة أي اعتماديات npm جديدة.

## لماذا تم تعديل الخطة؟

تمت تجربة Tiptap أولًا، لكن جهاز التطوير أظهر خطأ من npm نفسه أثناء `npm install`، لذلك تم اعتماد حل آمن لا يحتاج تثبيت حزم جديدة، ويزيل الاعتماد على Summernote/jQuery ويحافظ على طريقة الحفظ الحالية.

## ما يحافظ عليه المحرر الجديد

- حفظ المحتوى كـ HTML في نفس الحقل `content`.
- دعم العناوين والفقرات والقوائم والروابط والجداول البسيطة والمحاذاة.
- رفع الصور داخل المحتوى وإدراجها كـ `<img>`.
- رفع الملفات داخل المحتوى وإدراجها كرابط تحميل.
- عدم الحاجة لأي تعديل في الباك اند.

## ملاحظة

المحرر لا يعتمد على `eval` ولا على jQuery ولا Summernote. عند اكتمال اختبارات الإنتاج يمكن إزالة أي استثناءات CSP خاصة بالمحرر القديم.
