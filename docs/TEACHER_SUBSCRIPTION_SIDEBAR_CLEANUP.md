# تنظيم القائمة الجانبية لاشتراكات المعلمين

## القرار
تم جعل كل صفحات اشتراكات المعلمين تحت قسم واحد مستقل:

- اشتراكات المعلمين

## المسارات داخل هذا القسم
- لوحة إدارة الاشتراكات: /dashboard/teacher-subscriptions
- طلبات الاشتراك: /dashboard/teacher-subscriptions/orders
- خزنة ملفات Premium: /dashboard/teacher-subscriptions/premium-files
- المعلمون المشتركون: /dashboard/teacher-subscriptions/teachers
- الاشتراكات: /dashboard/teacher-subscriptions/subscriptions
- أجهزة المعلمين: /dashboard/teacher-subscriptions/devices
- تحميلات Premium: /dashboard/teacher-subscriptions/downloads
- عمليات AI: /dashboard/teacher-subscriptions/ai-generations

## ما تم إزالته
تم حذف تكرار روابط اشتراكات المعلمين من قسم:
- إدارة المستخدمين

بقي قسم إدارة المستخدمين خاصًا فقط بـ:
- المستخدمين
- الأدوار
- الصلاحيات
