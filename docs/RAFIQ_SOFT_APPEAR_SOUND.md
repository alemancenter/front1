تم إضافة صوت نقر خفيف لرفيق المنصة:
- يعمل عند ظهور التلميح لأول مرة في الصفحة الرئيسية أو صفحات login/register/signup.
- يعمل مرة واحدة لكل مسار ويحفظ ذلك في localStorage.
- الصوت مولد عبر Web Audio API ولا يحتاج ملف صوت خارجي.
- إذا منع المتصفح التشغيل قبل أول تفاعل، يتم تجاهل ذلك بدون أي خطأ.
- عند ضغط المستخدم على رفيق المنصة يتم تشغيل صوت خفيف أيضًا لأنه تفاعل مباشر ومسموح غالبًا.
