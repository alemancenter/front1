# Chatbot V4 Frontend

The chatbot widget now understands the AI metadata returned by the backend:

- `ai_used`
- `ai_model`

When the backend uses the guarded AI layer, the widget displays a small badge: "رد محسّن بالذكاء الاصطناعي".

No raw AI links are rendered. The UI continues to render only safe `actions` and `links` generated by the backend.
