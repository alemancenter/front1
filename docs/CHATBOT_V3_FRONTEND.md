# Chatbot V3 Frontend

## ما تم تطويره

- تحديث رسالة الترحيب والاقتراحات الأولى.
- إضافة زر بدء محادثة جديدة لإعادة ضبط السياق والجلسة.
- عرض عدد نتائج البحث المقترحة داخل رسالة المساعد.
- تحسين رسالة الانتظار لتوضح أن السؤال يتم تحليله.
- دعم أزرار الروابط والرسائل القادمة من الباك اند.
